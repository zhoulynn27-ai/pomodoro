# 🍅 Pomodoro Timer - 番茄钟

一款简约现代风格的桌面番茄钟应用，基于 PySide6 (Qt for Python) 构建。

## ✨ 功能特性

- **三种模式** — 专注（25分钟）、短休息（5分钟）、长休息（15分钟）
- **圆形进度条** — 渐变色圆环实时显示倒计时进度
- **自动切换** — 工作结束后自动切换到休息模式，每 4 个番茄后自动进入长休息
- **番茄计数** — 实时统计已完成的番茄数量
- **声音提醒** — 倒计时结束时播放系统提示音
- **无边框窗口** — 可拖拽的深色主题界面

## 🛠️ 技术栈

- **Python 3.11+**
- **PySide6** — Qt for Python 官方绑定
- **QPainter** — 自定义绘制圆形进度条

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install PySide6
```

### 2. 运行应用

```bash
python pomodoro.py
```

## 📖 使用方法

1. 点击 **▶ 开始** 启动 25 分钟专注倒计时
2. 专注结束后自动切换到休息模式
3. 每完成 4 个番茄，自动进入 15 分钟长休息
4. 随时可以 **⏸ 暂停** 或 **↺ 重置** 倒计时
5. 点击顶部 **专注/短休/长休** 按钮可手动切换模式
6. 拖拽窗口即可移动位置
